VM nativa v2.0

Programe em portugues. Execute em qualquer lugar.

BRZ e uma linguagem com sintaxe em portugues e VM nativa escrita em C. Zero dependencias externas — instale e rode. Sem Java, sem Python, sem nada.

exemplo.brz 
// Classes com heranca
classe Animal {
    iniciar(nome, som) {
        este.nome = nome
        este.som = som
    }

    falar() {
        mostrar(este.nome + " faz " + este.som)
    }
}

classe Cachorro herda Animal {
    iniciar(nome) {
        super.iniciar(nome, "Au au!")
    }
}

var rex = Cachorro("Rex")
rex.falar()
Recursos

Tudo o que voce precisa numa linguagem.

VM nativa, garbage collector, closures, classes com heranca e sintaxe 100% em portugues. Nada fica de fora.

Sintaxe em portugues

Palavras-chave como se, enquanto, funcao, classe. Sem barreiras de idioma para aprender logica.

VM nativa em C

Maquina virtual completa com bytecode, stack-based execution e zero dependencias. ~5.200 linhas de C99 puro.

Garbage collector

Gerenciamento automatico de memoria com mark-and-sweep GC e string interning. Voce nunca precisa se preocupar com alocacao.

Closures e funcoes

Funcoes de primeira classe, closures que capturam variaveis, recursao. Tudo funciona como voce espera.

Classes e heranca

Orientacao a objetos completa. Construtores com iniciar, metodos, atributos, heranca com herda e super.

Instala em segundos

Um npm install -g brzlang e pronto. Compila do fonte automaticamente. Nenhuma dependencia externa necessaria.

Como funciona

Codigo fonte direto para execucao.

A VM BRZ compila seu codigo para bytecode e executa imediatamente. Sem arquivos intermediarios, sem compilador externo.

Pipeline da VM

Seu codigo passa por quatro etapas em milissegundos — do texto fonte ate a execucao.

  • Scanner tokeniza o fonte .brz em portugues
  • Compiler (Pratt parser) emite bytecode de 45 opcodes
  • VM stack-based executa o bytecode diretamente
  • GC mark-and-sweep gerencia a memoria automaticamente
  • Tudo em ~5.200 linhas de C99 puro

funcao fib(n) {

    retorne fib(n - 1) + fib(n - 2)

Arquitetura

VM completa. Escrita do zero em C.

Nao e um wrapper. Nao usa bibliotecas externas. Cada componente foi construido do zero.

Scanner
.brz → tokens

Analise lexica com todas as palavras-chave em portugues. Rapido e preciso.

Compiler
tokens → bytecode

Pratt parser single-pass que emite bytecode diretamente. 45 opcodes.

VM
bytecode → execucao

Maquina virtual stack-based com dispatch loop otimizado. Rapida e leve.

GC
memoria automatica

Mark-and-sweep com string interning. Gerencia objetos sem intervencao.
