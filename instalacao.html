Instalacao

Cinco minutos para rodar seu primeiro programa.

BRZ precisa de Java 17 ou superior. O resto e um unico comando.

0

Pre-requisitos

Voce precisa ter o Java JDK 17 (ou superior) instalado. Verifique com:

$ java -version

Se nao tiver, baixe em adoptium.net ou instale via gerenciador de pacotes:

$ brew install openjdk@17 macOS
$ sudo apt install openjdk-17-jdk Ubuntu/Debian
1

Via npm (recomendado)

Se voce usa Node.js, esta e a forma mais simples:

$ npm install -g brzlang copiar

Apos a instalacao, o comando brz fica disponivel globalmente no terminal.

2

Via pip

Para quem prefere o ecossistema Python:

$ pip install brzlang copiar
3

Build a partir do fonte

Clone o repositorio e compile com Maven:

$ git clone https://github.com/DornelassDev/brz.git
$ cd brz
$ mvn package -DskipTests

O JAR executavel sera gerado em target/brz.jar. Depois, use o wrapper script:

$ ./brz rodar exemplos/01_ola_mundo.brz

Para tornar o comando global, adicione o diretorio ao PATH ou copie o script para /usr/local/bin.

4

Extensao VS Code

Instale a extensao para ter syntax highlighting, snippets e execucao integrada:

$ code --install-extension brz-lang copiar

Ou busque "BRZ Lang" na aba de extensoes do VS Code. A extensao oferece:

  • Coloracao de sintaxe completa
  • Mais de 30 snippets
  • Execucao com F5
  • Compilacao com Ctrl+Shift+B
  • Verificacao automatica ao salvar
  • Documentacao ao passar o mouse
5

Verificar a instalacao

Confirme que tudo esta funcionando:

$ brz versao

Voce deve ver algo como:

[BRZ] BRZ Lang v1.0.0
6

Primeiro programa

Crie um arquivo chamado programa.brz com o conteudo:

// Meu primeiro programa BRZ
numero a = 10
numero b = 20
mostrar("A soma e: " + (a + b))

Execute:

$ brz rodar programa.brz

Ou gere o equivalente em Python:

$ brz compilar programa.brz --para python
7

Criar um projeto

Use o comando brz novo para gerar a estrutura de um projeto:

$ brz novo projeto MeuApp

Isso cria:

MeuApp/
  src/
    principal.brz
  build/
  README.md
