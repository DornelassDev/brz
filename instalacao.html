Instalacao

Um minuto para rodar seu primeiro programa.

BRZ tem VM nativa — zero dependencias externas. Instale e programe.

0

Pre-requisitos

Voce precisa de Node.js (para instalar via npm) e um compilador C (gcc ou clang). A maioria dos sistemas ja tem:

$ node --version
$ cc --version

macOS: O compilador ja vem instalado (clang via Xcode Command Line Tools). Se nao tiver: xcode-select --install

Linux: A maioria das distros ja tem gcc. Se nao: sudo apt install build-essential

Nao precisa de Java. A VM BRZ e compilada do fonte C durante a instalacao. Zero dependencias externas.

1

Via npm (recomendado)

A forma mais simples. Instala e compila a VM automaticamente:

$ npm install -g brzlang copiar

O pacote inclui o codigo-fonte da VM em C. Durante a instalacao, ele detecta seu compilador (gcc/clang) e compila o binario nativo automaticamente. Apos instalar, o comando brz fica disponivel globalmente.

2

Build a partir do fonte

Clone o repositorio e compile com make:

$ git clone https://github.com/DornelassDev/brz.git
$ cd brz/vm
$ make

O binario sera gerado em vm/bin/brz. Teste com:

$ ./bin/brz ../exemplos/01_ola_mundo.brz

Para tornar global, copie o binario para o PATH:

$ sudo cp bin/brz /usr/local/bin/brz
3

Extensao VS Code

Instale a extensao para ter syntax highlighting, snippets e execucao integrada:

$ code --install-extension brz-lang copiar

Ou busque "BRZ Lang" na aba de extensoes do VS Code. A extensao oferece:

  • Coloracao de sintaxe completa
  • Mais de 30 snippets
  • Execucao com F5
  • Verificacao automatica ao salvar
  • Documentacao ao passar o mouse
4

Verificar a instalacao

Teste com um codigo inline:

$ brz -c 'mostrar("BRZ funcionando!")'

Voce deve ver:

BRZ funcionando!
5

Primeiro programa

Crie um arquivo chamado programa.brz com o conteudo:

// Meu primeiro programa BRZ
var a = 10
var b = 20
mostrar("A soma e: " + (a + b))

Execute:

$ brz programa.brz

Ou use o REPL interativo:

$ brz
brz

Linguagem de programacao em portugues. Feita no Brasil.
