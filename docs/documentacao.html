Documentacao da Linguagem BRZ

Referencia completa de sintaxe, tipos, estruturas e comandos.

Introducao

BRZ e uma linguagem de programacao com sintaxe inteiramente em portugues. Ela foi projetada para ser acessivel a falantes nativos de portugues sem sacrificar funcionalidade. O codigo BRZ e transpilado para Java, Python, C ou C++, gerando codigo limpo e executavel.

Um arquivo BRZ tem a extensao .brz e e executado pelo compilador via linha de comando:

$ brz rodar meuPrograma.brz

Ou transpilado para outra linguagem:

$ brz compilar meuPrograma.brz --para python

Variaveis

Variaveis sao declaradas com o tipo seguido do nome e, opcionalmente, um valor inicial.

numero idade = 25
decimal salario = 3500.50
texto nome = "Carlos"
logico ativo = verdadeiro
caractere letra = 'A'

Atribuicao composta tambem e suportada:

idade += 1
salario *= 1.1
nome = "Novo nome"

Use constante para declarar valores que nao podem ser reatribuidos.

Condicionais

Se / Senao se / Senao

se (nota >= 90) {
    mostrar("A")
} senao se (nota >= 80) {
    mostrar("B")
} senao se (nota >= 70) {
    mostrar("C")
} senao {
    mostrar("F")
}

Expressao ternaria

texto resultado = (idade >= 18) ? "Adulto" : "Menor"

Lacos de repeticao

Para

para (numero i = 0; i < 10; i++) {
    mostrar(i)
}

Para cada

lista texto nomes = ["Ana", "Bruno", "Clara"]
para cada (nome em nomes) {
    mostrar(nome)
}

Enquanto

numero n = 10
enquanto (n > 0) {
    mostrar(n)
    n -= 1
}

Faca-enquanto

numero tentativas = 0
faca {
    tentativas += 1
    mostrar("Tentativa " + tentativas)
} enquanto (tentativas < 3)

Controle: pare e continue

para (numero i = 0; i < 20; i++) {
    se (i % 2 == 0) { continue }
    se (i > 15) { pare }
    mostrar(i)
}

Escolha (switch)

escolha (opcao) {
    caso "salvar": {
        mostrar("Salvando...")
    }
    caso "sair": {
        mostrar("Saindo...")
    }
    padrao: {
        mostrar("Opcao invalida")
    }
}

Funcoes

Funcoes sao declaradas com a palavra-chave funcao, seguida do tipo de retorno, nome e parametros.

funcao numero somar(numero a, numero b) {
    retorne a + b
}

funcao vazio saudacao(texto nome) {
    mostrar("Ola, " + nome)
}

// Chamada
numero total = somar(10, 20)
saudacao("Maria")

Funcoes podem ser recursivas:

funcao numero fibonacci(numero n) {
    se (n <= 0) { retorne 0 }
    se (n == 1) { retorne 1 }
    retorne fibonacci(n - 1) + fibonacci(n - 2)
}

Classes

Classes suportam atributos, construtores, metodos e heranca.

classe Pessoa {
    texto nome
    numero idade

    construtor(texto nome, numero idade) {
        este.nome = nome
        este.idade = idade
    }

    funcao texto apresentar() {
        retorne "Sou " + este.nome + ", " + este.idade + " anos."
    }
}

Instanciacao

Pessoa p = novo Pessoa("Joao", 30)
mostrar(p.apresentar())
mostrar(p.nome)

Heranca

classe Aluno estende Pessoa {
    decimal media

    construtor(texto nome, numero idade, decimal media) {
        super(nome, idade)
        este.media = media
    }

    funcao logico aprovado() {
        retorne este.media >= 7.0
    }
}

Listas

lista numero notas = [8, 7, 9, 10, 6]

// Acesso por indice
mostrar(notas[0])

// Adicionar e remover
adicionar(notas, 5)
remover(notas, 7)

// Tamanho
mostrar(tamanho(notas))

// Percorrer
para cada (nota em notas) {
    mostrar(nota)
}

CRUD

BRZ possui operacoes CRUD como parte da sintaxe. Elas geram codigo de armazenamento na linguagem alvo.

// Criar
criar("Produto", {nome: "Mouse", preco: 49.90})

// Listar todos
listar("Produto")

// Atualizar por ID
atualizar("Produto", 1, {preco: 59.90})

// Deletar por ID
deletar("Produto", 1)

Na transpilacao para Java, o CRUD gera uma estrutura baseada em HashMap. Em Python, usa dicionarios. No alvo C/C++, gera comentarios indicando a necessidade de implementacao externa.

Tratamento de erros

tente {
    numero resultado = dividir(10, 0)
} pegue (erro) {
    mostrar("Erro capturado: " + erro)
} finalmente {
    mostrar("Bloco finalmente executou.")
}

Para lancar erros manualmente:

funcao vazio validar(numero valor) {
    se (valor < 0) {
        lance("Valor nao pode ser negativo")
    }
}

Transpilacao

O comando brz compilar gera codigo na linguagem escolhida:

$ brz compilar programa.brz --para java
$ brz compilar programa.brz --para python
$ brz compilar programa.brz --para c
$ brz compilar programa.brz --para cpp

O flag --para aceita os valores java, python (ou py), c e cpp. Se omitido, o padrao e java.

Para executar diretamente sem gerar arquivo intermediario:

$ brz rodar programa.brz

Isso transpila para Java, compila com javac, executa e limpa os arquivos temporarios automaticamente.
