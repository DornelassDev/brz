Linguagem de programacao v1.0

Programe em portugues. Compile para qualquer lugar.

BRZ e uma linguagem com sintaxe em portugues que transpila para Java, Python, C e C++. Escreva codigo na sua lingua e gere projetos reais.

Comecar agora Ler documentacao
exemplo.brz 
// Sistema de notas
classe Aluno {
    texto nome
    decimal media

    construtor(texto nome, decimal media) {
        este.nome = nome
        este.media = media
    }

    funcao texto situacao() {
        se (este.media >= 7.0) { retorne "Aprovado" }
        retorne "Reprovado"
    }
}

Aluno aluno = novo Aluno("Maria", 8.5)
mostrar(aluno.situacao())
Recursos

Tudo o que voce precisa numa linguagem.

Tipos, classes, funcoes, tratamento de erros, CRUD nativo e transpilacao para quatro linguagens. Nada fica de fora.

Sintaxe em portugues

Palavras-chave como se, para, funcao, classe. Sem barreiras de idioma para aprender logica.

Transpilacao multi-alvo

Um unico arquivo .brz gera codigo Java, Python, C ou C++. Escolha o alvo com uma flag.

CRUD nativo

Operacoes criar, listar, atualizar e deletar fazem parte da sintaxe da linguagem.

Tipagem forte

Tipos numero, decimal, texto, logico e caractere. Erros pegos antes de rodar.

Classes e heranca

Programacao orientada a objetos completa. Construtores, metodos, atributos e heranca com estende.

Tratamento de erros

Blocos tente/pegue/finalmente e lance para controle total de excecoes.

Como funciona

Escreva uma vez, gere em quatro linguagens.

O compilador BRZ le seu codigo, constroi uma arvore sintatica e gera codigo nativo na linguagem que voce escolher.

Pipeline de compilacao

O processo e simples: o Lexer tokeniza seu codigo, o Parser monta a AST, e o Gerador produz o codigo final. Tudo em milissegundos.

  • Lexer analisa e tokeniza o fonte .brz
  • Parser monta arvore sintatica abstrata (AST)
  • Gerador percorre a AST via Visitor pattern
  • Codigo nativo gerado com tipos corretos
  • Suporte a Java, Python 3, C (C99) e C++17
entrada.brz 
funcao numero fatorial(numero n) {
    se (n <= 1) {
        retorne 1
    }
    retorne n * fatorial(n - 1)
}

mostrar(fatorial(10))
saida.java 
public class Entrada {
    static int fatorial(int n) {
        if (n <= 1) {
            return 1;
        }
        return n * fatorial(n - 1);
    }

    public static void main(String[] args) {
        System.out.println(fatorial(10));
    }
}
Alvos

Quatro linguagens. Um comando.

Transpile para a linguagem que seu projeto exige. O codigo gerado e limpo e idiomatico.

Java
.java

Gera classes Java completas com metodo main, tipos corretos e imports.

Python
.py

Codigo Python 3 idiomatico com indentacao correta e guard de main.

C
.c

Codigo C99 com headers corretos, structs para classes e printf/scanf.

C++
.cpp

C++17 com std::string, std::vector, iostream e classes nativas.

Instalacao

Comece em segundos.

Instale com um unico comando. Funciona em macOS, Linux e Windows.

npm
$ npm install -g brzlang copiar
pip
$ pip install brzlang copiar
Depois
$ brz rodar meuPrograma.brz copiar
